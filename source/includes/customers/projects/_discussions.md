This endpoint retrieves the discussions in your project.

### HTTP Request

`GET <%= api_url('/projects/<PROJECT_ID>/discussions') %>`

### URL Parameters

Parameter | Description
--------- | -----------
PROJECT_ID | The ID of the project
page | A specific page of results.
